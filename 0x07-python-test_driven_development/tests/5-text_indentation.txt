===========================
How to use text_indentation
===========================

"""
text_indentation print a square pattern
to stdout. 
"""

>>> text_indentation = __import__('5-text_indentation').text_indentation


test 1:

>>> text_indentation("Linea 1. Linea 2?, linea3:")
Linea 1.
<BLANKLINE>
Linea 2?
<BLANKLINE>
, linea3:
<BLANKLINE>
